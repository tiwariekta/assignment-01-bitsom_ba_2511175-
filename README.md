# 📊 BITSom BA Assignment — Data Engineering & Analytics

---

## 🧾 Overview

This assignment demonstrates end-to-end understanding of modern data systems, including:

- Relational Databases (RDBMS)
- NoSQL Databases
- Data Warehousing
- Vector Databases
- Data Lakes & DuckDB
- Scalable Data Architecture Design

> 📌 This project follows best practices in data modeling, ETL design, and modern analytics architecture.

---

## 📂 Repository Structure

---

## 🧩 Part 1 — Relational Databases

- Identified **data anomalies** (insert, update, delete)
- Designed a **3NF normalized schema**
- Implemented SQL queries for:
  - Customer analysis
  - Product performance
  - Sales insights

➡️ Focus: **Data integrity & normalization**

---

## 📦 Part 2 — NoSQL Databases

- Designed flexible **MongoDB documents** for:
  - Electronics
  - Clothing
  - Grocery
- Implemented CRUD operations and indexing
- Compared **RDBMS vs NoSQL** using ACID, BASE, and CAP theorem

➡️ Focus: **Schema flexibility & scalability**

---

## 🏗️ Part 3 — Data Warehouse

- Built a **Star Schema**:
  - Fact table: `fact_sales`
  - Dimensions: `dim_date`, `dim_store`, `dim_product`
- Performed data cleaning & ETL transformations
- Wrote analytical queries for:
  - Revenue trends
  - Store performance
  - Monthly analysis

➡️ Focus: **Business intelligence & reporting**

---

## 🧠 Part 4 — Vector Databases

- Generated embeddings using **Sentence Transformers**
- Built a **cosine similarity matrix**
- Performed semantic search using a query sentence

➡️ Focus: **Semantic understanding & AI-driven search**

---

## 🗃️ Part 5 — Data Lake & DuckDB

- Queried data directly from:
  - CSV (`customers.csv`)
  - JSON (`orders.json`)
  - Parquet (`products.parquet`)
- Performed cross-format joins using DuckDB
- Demonstrated schema-on-read approach

➡️ Focus: **Modern data lake querying**

---

## 🚀 Part 6 — Capstone Design

- Designed a scalable **data architecture**
- Defined:
  - Storage systems (OLTP, NoSQL, Lakehouse)
  - OLTP vs OLAP boundaries
  - Trade-offs and mitigation strategies

➡️ Focus: **Real-world system design**
